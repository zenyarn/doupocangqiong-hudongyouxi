# 游戏数值系统说明

## 一、四大核心数值

### 1. 血量 ❤️

| 属性 | 说明 |
|------|------|
| 范围 | 0 ~ 100 |
| 初始值 | 80 |
| 作用 | **生存判定**，血量归零即死亡 |

**影响方式**：
- ⬇️ 战斗受伤、冒险失败、修炼反噬
- ⬆️ 服用丹药、接受治疗、安全选择

**与结局关系**：血量归零 → 触发「陨落」结局

---

### 2. 斗气 ⚡

| 属性 | 说明 |
|------|------|
| 范围 | 0 ~ 100 |
| 初始值 | 20 |
| 作用 | **修炼资源**，蓄满后转化为修为 |

**转化规则**：
```
斗气 ≥ 100 → 斗气清零 + 修为 +15
```

**影响方式**：
- ⬇️ 战斗消耗、冒险行动
- ⬆️ 修炼、突破、服用丹药

**UI提示**：斗气 ≥ 90 时高亮闪烁，提示即将转化

---

### 3. 丹药 💊

| 属性 | 说明 |
|------|------|
| 范围 | 0 ~ 100 |
| 初始值 | 10 |
| 作用 | **资源储备**，用于恢复血量或斗气 |

**获取方式**：
- 探索发现
- 击败敌人
- NPC赠送
- 购买/交易

**消耗方式**：
- 部分选项需要消耗丹药
- 换取修为或斗气

---

### 4. 修为 ✨

| 属性 | 说明 |
|------|------|
| 范围 | 0 ~ 无上限 |
| 初始值 | 0 |
| 作用 | **段位晋升 + 结局判定** |

**获取方式**：
- 修炼突破（主要来源）
- 斗气转化（+15/次）
- 选项奖励
- 随机事件

---

## 四、结局判定流程

```
游戏结束条件：
├── 血量 ≤ 0 → 「陨落」结局
└── 到达结束节点 → 根据修为判定结局
                  │
                  ├── 修为 < 30 → 凡人之路
                  ├── 修为 < 150 → 强者初成
                  ├── 修为 < 400 → 王者之风
                  ├── 修为 < 900 → 宗师之名
                  ├── 修为 < 1800 → 尊者传说
                  ├── 修为 < 3500 → 圣者之路
                  └── 修为 ≥ 3500 → 炎帝传说
```

## 五、数值平衡建议

### 当前设计理念
1. **血量**：唯一负面结局触发条件，强调生存策略
2. **斗气**：短期目标（蓄满转化），提供持续正向反馈
3. **丹药**：资源管理维度，关键抉择
4. **修为**：长期目标，决定最终成就

### 平衡要点
- 斗气转化提供稳定的修为来源
- 避免单一数值决定一切
- 不同选择有不同风险收益
